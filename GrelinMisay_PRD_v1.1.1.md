## 一、产品概述

### 1.1 产品背景

现代年轻人面临工作压力、生活焦虑、健康管理四大难题：
- **时间碎片化**：工作与生活边界模糊，事务繁杂难以统筹
- **目标难坚持**：个人目标孤立无援，缺乏监督动力
- **健身门槛高**：专业健身APP复杂难用，缺乏简洁方案
- **社交孤独感**：朋友圈信息过载，难找到同频好友

GrelinMisay致力于打造一款**简洁、易用、社交化**的全能自律APP。

### 1.2 产品定位

| 维度 | 描述 |
|------|------|
| **产品名称** | GrelinMisay |
| **产品类型** | 跨平台自律管理APP（H5 + APK） |
| **核心定位** | 健身训练 + 目标管理 + 日历日程 + AI助手 |
| **目标用户** | 18-35岁学生/职场人士，有自律需求 |
| **核心差异** | 简洁够用 + 社交监督 + AI智能辅助 + 多场景Agent |

## 三、功能需求详述

### 3.1 账号系统（v1.1.1 增强）

#### 3.1.1 登录注册

| 功能 | 交互逻辑 | 必填 |
|------|---------|------|
| 手机号登录 | 输入手机号 → 获取验证码 → 填写验证码 → 登录 | 是 |
| 密码登录 | 手机号 + 密码 → 登录 | 是 |
| 注册账号 | 手机号 + 验证码 + 设置密码 + 昵称 | 是 |
| 退出登录 | 确认弹窗 → 清除Token → 跳转登录页 | - |

**v1.1.1 新增安全机制**：

| 安全措施 | 说明 |
|---------|------|
| **验证码尝试限制** | 输错5次后验证码自动失效 |
| **发送频率限制** | 同一手机号60秒内只能发一次 |
| **IP限制** | 同一IP每天限制发送次数 |
| **JWT Token** | 支持多Worker部署，Token跨进程共享 |
| **Token刷新** | 30天有效期，支持无感刷新 |

#### 3.1.2 个人资料

| 字段 | 编辑方式 | 必填 |
|------|---------|------|
| 头像 | 相册选择/拍照上传 | 必填 |
| 昵称 | 文本输入，限20字符 | 必填 |
| 个人简介 | 文本输入，限200字符 | 选填 |
| 性别 | 单选：男/女/保密 | 选填 |
