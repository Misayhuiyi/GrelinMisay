## 一、产品概述

### 1.1 产品背景

现代年轻人面临工作压力、生活焦虑、健康管理四大难题：
- **时间碎片化**：工作与生活边界模糊，事务繁杂难以统筹
- **目标难坚持**：个人目标孤立无援，缺乏监督动力
- **健身门槛高**：专业健身APP复杂难用，缺乏简洁方案
- **社交孤独感**：朋友圈信息过载，难找到同频好友

GrelinMisay致力于打造一款**简洁、易用、社交化**的全能自律APP。

### 1.2 产品定位

| 维度 | 描述 |
|------|------|
| **产品名称** | GrelinMisay |
| **产品类型** | 跨平台自律管理APP（H5 + APK） |
| **核心定位** | 健身训练 + 目标管理 + 日历日程 + AI助手 |
| **目标用户** | 18-35岁学生/职场人士，有自律需求 |
| **核心差异** | 简洁够用 + 社交监督 + AI智能辅助 |

### 1.3 目标用户画像

```
┌─────────────────────────────────────────────────────────────┐
│                     目标用户画像                             │
├─────────────────────────────────────────────────────────────┤
│  用户名: 小明 (典型代表)                                     │
│  年龄: 25岁 / 职场人士 / 单身                               │
│  职业: 互联网公司产品经理                                    │
│                                                             │
│  痛点:                                                      │
│  • 想健身但不知道如何计划                                     │
│  • 立的flag总是倒，没人监督                                  │
│  • 日程和工作事项分散在多个APP                               │
│  • 想找人一起互相监督                                        │
│                                                             │
│  行为特征:                                                  │
│  • 每天通勤刷手机1-2小时                                     │
│  • 愿意为高效工具付费                                        │
│  • 喜欢社交互动和被认可                                      │
│  • 对AI功能有好奇心                                          │
└─────────────────────────────────────────────────────────────┘
```

### 1.4 核心价值主张

| 功能模块 | 价值主张 | 用户收益 |
|---------|---------|---------|
| **健身训练** | 专业但不复杂 | 5分钟记录每次训练，数据清晰可见 |
| **目标管理** | 目标+好友监督 | 不再孤军奋战，好友互相督促 |
| **日历日程** | 统一视图 | 所有事务一个APP全搞定 |
| **AI助手** | 智能规划 | 健身计划、目标拆分一句话搞定 |

## 三、功能需求详述

### 3.1 账号系统

#### 3.1.1 登录注册

| 功能 | 交互逻辑 | 必填 |
|------|---------|------|
| 手机号登录 | 输入手机号 → 获取验证码 → 填写验证码 → 登录 | 是 |
| 密码登录 | 手机号 + 密码 → 登录 | 是 |
| 注册账号 | 手机号 + 验证码 + 设置密码 + 昵称 | 是 |
| 退出登录 | 确认弹窗 → 清除Token → 跳转登录页 | - |

**验证码规则**：60秒倒计时，防刷机制

#### 3.1.2 个人资料

| 字段 | 编辑方式 | 必填 |
|------|---------|------|
| 头像 | 相册选择/拍照上传 | 必填 |
| 昵称 | 文本输入，限20字符 | 必填 |
| 个人简介 | 文本输入，限200字符 | 选填 |
| 性别 | 单选：男/女/保密 | 选填 |
| 生日 | 日期选择器 | 选填 |

### 3.2 首页（日历总览）

#### 3.2.1 日历组件

| 视图 | 展示内容 | 交互 |
|------|---------|------|
| 周视图 | 7天日期 + 顶部切换按钮 | 左右滑动切周 |
| 日详情 | 当日所有事务时间线 | 点击日期查看详情 |

**日历聚合内容**：

| 类型 | 内容 | 颜色 |
|------|------|------|
| 纪念类 | 纪念日、胶囊解封日期 | 🔴 红色 |
| 任务类 | 个人待办、共同目标截止日 | 🟢 绿色 |
| 健康类 | 健身打卡、饮食记录 | 🔵 蓝色 |
| 社交类 | 动态更新、互动消息 | 🟡 黄色 |

#### 3.2.2 今日事务

- 按时间顺序展示当日所有事务
- 支持快捷操作（勾选完成、跳转详情）

#### 3.2.3 快捷入口

| 入口 | 图标 | 跳转 |
|------|------|------|
| 打卡 | ✓ | 快速打卡 |
| 训练 | 💪 | 开始训练 |
| 记录 | 📝 | 新建待办 |
| AI | 🤖 | AI助手 |

### 3.3 目标管理系统

#### 3.3.1 个人Todo

| 功能 | 描述 |
|------|------|
| 创建任务 | 标题 + 截止时间 + 优先级（高/中/低）+ 标签 |
| 任务操作 | 打卡完成✓、标记未完成、归档 |
| 属性配置 | 优先级标签、进度百分比 |
| 快捷添加 | 首页快捷入口一键添加 |

**任务状态**：待办 → 进行中 → 已完成 / 已过期

#### 3.3.2 共同目标（社交核心）

| 功能 | 描述 |
|------|------|
| 创建目标 | 标题 + 描述 + 截止时间 + 目标类型 |
| 邀请好友 | 搜索手机号添加成员，最多10人 |
| 任务拆分 | 添加子任务 + 分配成员职责 |
| 进度管理 | 成员独立打卡 + 全员进度看板 |
| 完成率 | 自动计算，显示百分比和进度条 |

**共同目标流程**：
```
创建目标 → 邀请好友 → 好友接受 → 设置子任务 → 成员打卡 → 进度同步 → 完成
```

### 3.4 健身训练系统

#### 3.4.1 训练动作分类

| 大类 | 子类 | 说明 |
|------|------|------|
| **胸部** | 卧推、飞鸟、俯卧撑等 | 器械/徒手 |
| **背部** | 引体向上、划船等 | 器械/徒手 |
| **肩部** | 推举、侧平举等 | 器械/徒手 |
| **手臂** | 二头弯举、三头下压等 | 器械/徒手 |
| **腿臀** | 深蹲、硬拉、腿举等 | 器械/徒手 |
| **核心** | 卷腹、平板支撑等 | 徒手 |
| **有氧** | 跑步、骑行、椭圆机等 | 器械/GPS |

#### 3.4.2 单组训练参数

| 参数 | 配置 |
|------|------|
| 组数 | 1-20组 |
| 次数 | 支持"力竭"标记 |
| 重量 | kg（支持小数） |
| 组间间歇 | 秒 |
| 训练感受 | 😍累 / 😎一般 / 😐轻松 |
| 备注 | 自定义文字 |

**快捷操作**：
- 复刻历史整套训练
- 一键重量递增（+2.5kg）
- 组间休息倒计时

#### 3.4.3 训练记录

| 功能 | 说明 |
|------|------|
| 训练模板 | 保存/使用自定义模板 |
| 历史记录 | 按日期查询历史训练 |
| 数据统计 | 当日总训练容量、总组数、总时长 |

### 3.5 身体数据记录

| 录入项 | 频率 | 可视化 |
|--------|------|--------|
| 体重 | 每日 | 日/周/月趋势曲线 |
| 体脂率 | 每周 | 趋势曲线 |
| 围度 | 每周 | 变化对比图 |
| 照片 | 每周 | 相册时间线 |

### 3.6 AI智能助手

#### 3.6.1 AI能力矩阵

| 能力 | 功能描述 |
|------|---------|
| **日常聊天** | 日常闲聊、情绪疏导 |
| **健身计划** | 根据目标生成训练计划 |
| **目标拆分** | 拆解年度/月度目标为可执行步骤 |
| **训练建议** | 根据历史数据给出调整建议 |
| **文案生成** | 健身打卡文案、目标寄语 |

#### 3.6.2 对话示例

```
用户: 帮我制定一个减脂计划
AI:   [生成个性化减脂计划，包含饮食+训练建议]

用户: 我的目标是3个月减重10斤
AI:   [拆解为每月目标，每周训练计划]

用户: 今天练胸，80kg卧推5组力竭了
AI:   [记录训练数据，给出恢复建议]
```

| **核心功能** | 目标+训练+AI | 课程+社区 | 饮食+体重 | 精细化训练 |
